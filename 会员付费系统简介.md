# 数学证明导师 - 会员付费系统

## 项目概述

数学证明导师的会员付费系统，为用户提供灵活的会员等级和多种付费服务。

## 🎯 最终定价

| 会员等级 | 价格 | 特色 |
|---------|------|------|
| **月度会员** | ¥2/月 | 买杯奶茶都不够！ |
| **年度会员** ⭐ | ¥19.9/年 | 最划算，性价比之王 |
| **终身会员** | ¥39.9 | 一次付费永久使用 |

## 权益对比

| 功能 | 免费版 | 月度会员 | 年度会员 | 终身会员 |
|------|--------|----------|----------|----------|
| 题库访问 | 100题 | 500题 | 2000+题 | 2000+题 |
| OCR识别 | 每日5次 | 每日50次 | 无限次 | 无限次 |
| 分析深度 | 基础 | 深度+失分点 | 完整+知识图谱 | 完整+知识图谱 |
| 学习报告 | ❌ | 月度报告 | 详细报告 | 详细报告 |
| 题库扩充 | ❌ | ❌ | ✅ 4000+题 | ✅ 4000+题 |
| API权限 | ❌ | ❌ | ❌ | ✅ |
| 优先支持 | ❌ | ❌ | ❌ | ✅ |

## 快速启动

```bash
# 1. 初始化数据库
mysql -u root -p < database/membership_schema.sql

# 2. 安装依赖
pip install flask

# 3. 启动服务
python app.py
```

## 访问页面

- 会员购买页: http://localhost:5000/membership
- 价格对比页: http://localhost:5000/pricing

## 技术栈

- **后端**: Python Flask
- **数据库**: MySQL
- **支付**: 支付宝/微信支付/Stripe
- **前端**: HTML5 + CSS3 + JavaScript
